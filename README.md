# ClaudeRepo

This repository contains Salesforce Einstein AI / Agentforce code examples built with Claude Code.

## Overview

A collection of Salesforce DX (SFDX) project files demonstrating how to build AI-powered features using:
- Einstein Copilot Actions (Invocable Apex)
- Prompt Builder integrations
- Lightning Web Components (LWC)
- Agentforce custom actions

## Project Structure

- `force-app/` — Salesforce source metadata
- `manifest/` — Deployment manifests
- `scripts/` — Apex and SOQL scripts
